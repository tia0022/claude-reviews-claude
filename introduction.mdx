# Claude Reviews Claude

**When AI Reads Its Own Source Code**

17 architecture deep dives written by Claude, dissecting Claude Code v2.1.88 — 1,902 files, 477,439 lines of TypeScript.

---

## What Is This?

Claude Code is Anthropic's CLI for Claude. This project documents its internal architecture as analyzed by Claude itself — a unique exercise in AI self-reflection at a production engineering level.

The analysis covers every major subsystem, from the core query loop to the terminal UI, written as a series of 17 deep-dive chapters.
